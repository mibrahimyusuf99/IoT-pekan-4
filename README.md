IoT Monitoring dengan ESP32 dan ThingSpeak

Proyek ini merupakan implementasi dasar Internet of Things menggunakan Heltec ESP32 LoRa WiFi 
yang terhubung ke WiFi untuk melakukan monitoring suhu dan kelembapan. 
Data diperoleh dari sensor DHT22 (atau simulasi) dan dikirim secara berkala ke platform ThingSpeak menggunakan protokol HTTP.

Selain itu, sistem juga dilengkapi dengan kontrol LED sebagai indikator kondisi suhu tertentu. 
Proyek ini dikembangkan menggunakan Arduino IDE dan bertujuan untuk memahami dasar komunikasi data, 
integrasi sensor, serta pengiriman data ke cloud dalam sistem IoT.
